# SpacioPort - Flexible Workspaces for Indian Businesses

## About

SpacioPort is a modern web platform that connects businesses across India with flexible workspace solutions - virtual offices, coworking spaces, on-demand meeting rooms, and long-term leasing.

## Tech Stack

- React + TypeScript
- Vite
- Tailwind CSS
- shadcn/ui
- Node.js + Express + MongoDB (backend)

## Getting Started

### Frontend

```sh
npm install
npm run dev
```
### Backend

See [spacioport backend/README.md](spacioport backend/README.md) for full setup instructions.

## Project Structure

- `src/` — React frontend
- `backend/` — Node.js + Express API with MongoDB
